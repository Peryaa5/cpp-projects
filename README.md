# cpp-projects

Проекты на C++: алгоритмы, графический интерфейс, игры, консольные программы и практические задания.

Курсовой проект на C++/Windows Forms. Программа вычисляет наибольший общий делитель и наименьшее общее кратное нескольких чисел, обрабатывает ошибочный ввод и выводит результат через графический интерфейс.
