# cpp-projects

Проекты на C++: алгоритмы, графический интерфейс, игры, консольные программы и практические задания.

## Проекты

### 1. Игра на C++ / WinAPI

Игровой проект на C++ в Visual Studio с использованием WinAPI.  
Проект находится в разработке и постепенно дополняется.

**Ссылки:**
- [Папка проекта](winapi-game-cpp/)


### 2. Детективная игра на C++
Игровой проект на C++, в котором реализована детективная сюжетная игра с пользовательским сценарием, настройками и визуальной демонстрацией результата. Проект показывает навыки работы с логикой программы, структурированием кода, обработкой пользовательских действий.
**Ссылки:**
- [Папка проекта](./antifreeze-detective-game)

### 3. Калькулятор НОД и НОК на C++ с графическим интерфейсом
Проект на C++/Windows Forms. Программа вычисляет наибольший общий делитель и наименьшее общее кратное нескольких чисел. Проект демонстрирует работу с алгоритмом Евклида, графическим интерфейсом, обработкой ошибок ввода.
**Ссылки:**
- [Папка проекта](./gcd-lcm-calculator-cpp)
