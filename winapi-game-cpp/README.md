# Игра на C++ / WinAPI

Игровой проект на C++, созданный в Visual Studio.

Проект представляет собой Windows-приложение с использованием WinAPI, ресурсов приложения и отдельной логики игрового движка.

## Что реализовано

- создание Windows-приложения на C++;
- работа с проектом Visual Studio;
- использование `.sln` и `.vcxproj`;
- обработка основного окна приложения;
- подключение ресурсов приложения;
- работа с иконками и `.rc`-файлом;
- разделение логики на основные файлы `Main.cpp` и `Engine.cpp`;
- базовая структура игрового проекта.

## Основные файлы

```text
GAME.sln                 — файл решения Visual Studio
GAME/GAME.vcxproj        — файл проекта Visual Studio
GAME/Main.cpp            — основной файл приложения
GAME/Main.h              — заголовочный файл основного модуля
GAME/Engine.cpp          — логика игрового движка
GAME/Engine.h            — заголовочный файл игрового движка
GAME/Resource.h          — идентификаторы ресурсов
GAME/GAME.rc             — файл ресурсов Windows-приложения
GAME/GAME.ico            — иконка приложения
